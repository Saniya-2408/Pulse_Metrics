# Pulse_Metrics
Real-Time Sales Intelligence Dashboard PulseMetrics is an interactive Business Intelligence dashboard built with pure HTML, CSS, and JavaScript. It  displays monthly bar charts, and includes interactive filters and a what-if growth projector (0-50%). The project has 4 linked pages: Home Dashboard, Monthly Report, Product Pulse, and Signal Insights page with actionable BI recommendations.

This project demonstrates core BI analyst skills including data visualization, KPI tracking, interactive filtering, and business communication. No frameworks or backend required – just open index.html in any browser.
